4. Installationsanweisungen
WICHTIG: Dies ist eine Beta-Version und daher nicht für den Einsatz in Produktiv-Umgebungen gedacht!

Update-Hinweise:
  • Dies ist nur von Boardsolution v1.3x möglich. Falls Sie eine ältere Version haben, müssen Sie schrittweise updaten
  • Bitte sichern Sie das gesamte Board vorher das gesamte Board zu sichern! Nur um sicher zu gehen... (Dies ist eine Beta-Version!)
    "Das gesamte" heißt, dass Sie den Datenbankinhalt (Dies können Sie z.B. mit Boardsolution (Adminbereich -> Datenbanksicherung), PHPMyAdmin oder etwas ähnlichem tun) und alle Dateien des Boards sichern sollten.
  • Löschen Sie bitte zuerst die Dateien und Ordner der alten Version um Probleme zu vermeiden.
    Aber Vorsicht: Löschen Sie keine wichtigen Daten. Dazu gehören folgende Ordner (inkl. Inhalt):
    • dbsave/
    • images/avatars/
    • images/smileys/
    • uploads
    Und natürlich Dateien, die Sie selbst verändert haben.
    Bitte beachten Sie außerdem noch folgendes:
    • Falls Sie das Design eines Themes im Adminbereich verändert haben, wurde dadurch die Datei themes/<theme>/style.css verändert.
    • Falls Sie Templates im Adminbereich verändert haben, wurden diese in themes/<theme>/templates/ verändert.
    • Evtl. haben Sie auch Änderungen an der install/user_config.php vorgenommen.
  • Die Installation läuft genauso wie bei einer neuen Installation, abgesehen davon, dass Sie "Update" auswählen müssen.

Sie können das Script folgendermaßen installieren:
  1. Laden Sie alle Dateien auf Ihren Server hoch
  2. Rufen Sie die Datei "install.php" in Ihrem Browser auf. Diese wird Sie durch die Installation leiten.
  3. Folgen Sie der Installation
  4. Löschen Sie nun die "install.php"
  5. Wenn Sie vorhaben das Board in Ihre Seite mittels PHP einzubinden, kann es zu Probleme kommen. ( Im Zweifelsfall sehen Sie dies daran, dass ein Fehler wie "header already sent by ..." auftritt :-) ) Sie können dies folgendermaßen verhindern: Fügen Sie ganz am Anfang (WICHTIG! Dies muss wirklich vor allen anderen Sachen stehen) der Datei in welche Sie das Board einbinden (Falls diese auch eingebunden wird, muss es an die Datei von der alles ausgeht) folgendes ein:
    <?php
    ob_start();
    ?>

    und ganz am Ende dieser Datei fügen Sie folgendes ein:
    <?php
    ob_end_flush();
    ?>
    Wichtig: Das ganze muss nur beachtet werden, falls Sie Boardsolution per PHP einbinden, d.h. per include(), require(),include_once() oder require_once()
  6. Sie können sich jetzt mit dem in der Installation festgelegten Account ins Board einloggen und dort Boardsolution so einstellen, wie Sie es haben möchten.
  7. Bitte beachten Sie auch die Einstellungen in den Dateien "config/userdef.php", "config/bbcode.php" und "config/actions.php". Dort sind einige Detail-Einstellungen vorhanden, die dort alle (hoffentlich) gut und ausreichend erläutert sind.
  8. Falls Sie mit Boardsolution Probleme irgendeiner Form haben, dürfen Sie sich gerne in unserem Support-Board oder per Email an uns wenden.
  9. Das wars! Viel Spass mit Boardsolution :-)
