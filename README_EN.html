4. Install - instructions
IMPORTANT:If you have already Boardsolution v1.30 Beta1 or Beta2 installed please look at this topic to see how to update! In this case it is not necessary and possible to run the install.php!

Update-notices:
  • This is only possible from Boardsolution v1.2x. If you have an older version you have to update step by step
  • It is not required but I would recommend to backup the complete board in every case. Just to be sure...
    "Complete" means that you should save the database-content (you can do that for example with Boardsolution (Adminarea -> Database-backup), PHPMyAdmin or something similar) and save all files of the board.
  • Please delete at first all files and folders of the old version to prevent problems.
    But be carefull: Don't delete important data. These are the following folders (incl. content):
    • dbsave/
    • images/avatars/
    • images/smileys/
    • uploads
    And of course the files you have changed by yourself.
    Please also note the following:
    • If you have changed the design of a theme in the adminarea the file themes/<theme>/style.css has been changed.
    • If you have changed templates in the adminarea this has been done in themes/<theme>/templates/.
    • Perhaps you have also changed something in the install/user_config.php.
  • Besides the above mentioned things there is no different in the installation compared to a full installation except that you have to choose "Update"

You can install the script as follows:
  1. Upload all files to your server
  2. Call the install.php in your browser and follow the installation.
  3. Now delete the "install.php"
  4. If you want to include Boardsolution with PHP the script may cause trouble. ( If you don't know if this is true you will see this if an error like "header already sent by ..." appears :-) )
    You can prevent that by the following:
    Please put at the very beginning (This is REALLY important, otherwise this has no effect) of the file in which you include Boardsolution (if this file will be included, too, you have to put this in the "root-file" of your homepage) the following code:
    <?php
    ob_start();
    ?>

    and at the end of this file you integrate the following:
    <?php
    ob_end_flush();
    ?>
    Important: You only have to consider this if you include Boardsolution with PHP, that means with include(),require(),include_once() or require_once()
  5. Now you can login to the board with the username and password you've set in the installation and configure Boardsolution like you want to.
  6. Please notice the configurations in the files "config/userdef.php", "config/bbcode.php" and "config/actions.php". There are some detail-options which (I hope so) are explained very well.
  7. If you have problems with Boardsolution of any kind we will be satisfied if you use our support-board or write us an email.
  8. That was it! Have fun with Boardsolution :-)
